# BasicPulsePlayer (tvOS)

The tvOS variant of the Pulse sample (Apple TV only): Pulse playback through `BCOVTVPlayerView` with fixed content metadata and midroll positions (no scenario table or companion slot).

## Key files

| File | Responsibility |
|---|---|
| `BasicPulsePlayer/ViewController.swift` | tvOS player view, Pulse session provider, fixed metadata and midrolls |

Requires the manually-added INVIDI Pulse tvOS framework — see the [Pulse README](../) for setup.
