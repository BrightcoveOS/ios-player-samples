# BasicDAIPlayer (tvOS)

The tvOS variant of the DAI sample (Apple TV only): the Video Properties stream-request policy played through `BCOVTVPlayerView`.

## Key files

| File | Responsibility |
|---|---|
| `BasicDAIPlayer/ViewController.swift` | tvOS player view, DAI + FairPlay setup, Video Properties policy |

See the [DAI README](../) for shared setup.
